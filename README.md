# iGrafx Release Dashboard

## Description
The iGrafx Release Dashboard is a capstone project called iGrafx Release Dashboard, developed by the PSU Capstone Team. It integrates Jira, Sonar, and GitLab to provide a comprehensive project management solution. The dashboard includes a todo list management feature for each project, allowing users to track and manage their tasks efficiently. Additionally, the system sends email notifications whenever a todo list is modified, ensuring effective communication and collaboration among team members.

## Installation

To run the project, follow these steps:

1. Navigate to the `client` directory and run `npm i` to install the required dependencies.
2. Navigate to the `server` directory and run `npm i` to install the required dependencies.
3. Once the dependencies are installed, run `npm run dev` in both the `client` and `server` directories to start the development servers.

## Generating JSDoc

To generate JSDoc for the project, follow these steps:

1. Navigate to the `client` directory and run `npm run docs` to generate the JSDoc documentation for the client code.
2. Navigate to the `server` directory and run `npm run docs` to generate the JSDoc documentation for the server code.
3. Open the `index.html` file located in the `docs` folder of each respective directory to access the generated JSDoc documentation.

## Running Unit Tests

To run the unit tests, follow these steps:

1. Navigate to the `client` directory and run `npm run test` to execute the unit tests for the client code.
2. Navigate to the `server` directory and run `npm run test` to execute the unit tests for the server code.
